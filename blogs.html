How To Add A New Post

A simple system you can repeat

  1. Duplicate one of the existing `blog-*.html` files.
  2. Rename it with a new title, for example `blog-my-new-post.html`.
  3. Change the page title, article heading, date label, and paragraphs.
  4. Add a new card on this `blogs.html` page that links to the new file.
  5. Commit or upload the updated files to GitHub, and the website will refresh automatically.
